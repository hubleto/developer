# Hubleto Dev Hosting

Hubleto Dev Hosting gives developers a secure, production-like environment to build and test applications.
Catching issues early speeds up your launch time and helps you prototype faster. Plus, with VS Code Remote support,
you can build a new feature and instantly share a test link with customers. No live deployment needed.

<img src="../book/content/assets/images/dev_accounts.png" />

Through the Web UI, you can manage your assigned Hubleto accounts. At the moment, you can only create and delete the SSH
access to them.

<img src="../book/content/assets/images/dev_ssh.jpg" />
