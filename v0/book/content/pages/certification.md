# Certification

Become a certified Hubleto developer

## Level 1

The first level of certification ensures that the certified developer knows basics about Hubleto
development process, its architecture, built-in functionality and most commonly used software
design patterns like, e.g. routing, controllers or views.

The graduated student will learn how to:

  * prepare the development environment
  * deploy Hubleto to customer's server
  * create custom apps
  * define models, views and controllers
  * install custom app with default data

## Level 2

The second level follows up the first level and adds more detailed information about
more complex and useful design patterns or architectural components, like e.g. React UI or
advanced features of models.

The graduate student will learn how to:

  * customize look & feel of grids and forms
  * implement custom React UI components
  * implement model callbacks
  * integrate with other apps like *Calendars* or *Tasks*
  * various tips&tricks to boost effectivity

## Level 3

The third level adds even more detailed insight into Hubleto programming.
more comples and usefull design patterns or architectural components, like e.g. scheduled crons
or dependency injection.

The graduate student will learn how to:

  * write unit tests
  * create and configure scheduled jobs (crons)
  * trigger custom hooks and use existing ones
  * integrate with other apps like *Workflow*, *Documents* or *Mails*
  * implement REST API endpoints
